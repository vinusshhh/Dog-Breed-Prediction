# 🐶 Dog Breed Prediction

A deep learning application that predicts the breed of a dog from an uploaded image using a trained CNN model.

## 🔍 Features
- Upload dog image
- Predict dog breed
- Confidence score visualization
- Streamlit-based UI
- Works offline after setup

## 🛠 Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- PIL

## ▶️ How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
